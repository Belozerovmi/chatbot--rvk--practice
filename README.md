## О проекте

**Чат-бот для подбора персонала** представляет собой полноценное веб-приложение, разработанное в рамках производственной практики в компании **РВК-Воронеж** (ООО «РВК - Воронеж»). Система автоматизирует процесс первичного отбора кандидатов на вакантные должности, обеспечивая удобный интерфейс как для соискателей, так и для HR-специалистов.

**Архитектура решения** построена по принципу клиент-сервер с использованием REST API. Фронтенд реализован в двух вариантах: виджет для встраивания в корпоративный сайт и полноценная административная панель для управления данными. Бэкенд обеспечивает обработку запросов, валидацию данных, работу с файловой системой и взаимодействие с базой данных MySQL на нестандартном порту 3307.

**Ключевая особенность** проекта заключается в гибкости системы: вопросы к кандидату могут быть настроены как через код, так и через базу данных, что позволяет адаптировать процесс сбора информации под конкретные требования вакансии без изменения программного кода.

## Установка и запуск

### Шаг 1. Клонирование репозитория

```bash
git clone https://github.com/belozerovmi/ChatBot-RVK-belozerov.git
```

> **Важно:** Укажите ваше реальное имя пользователя GitHub вместо `ВАШ_USERNAME`.

### Шаг 2. Размещение в директории XAMPP

Скопируйте папку проекта в директорию:
```
C:\xampp\htdocs\ChatBot-RVK-belozerov\
```

### Шаг 3. Настройка порта MySQL

В файле `db.php` проверьте параметры подключения:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'chatbot_recruitment');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_PORT', '3307');   // ← Нестандартный порт
```

**Если ваш MySQL работает на стандартном порту 3306**, измените значение на:

```php
define('DB_PORT', '3306');
```

### Шаг 4. Запуск XAMPP Control Panel

1. Откройте **XAMPP Control Panel**
2. Нажмите **Start** напротив модуля **Apache**
3. Нажмите **Start** напротив модуля **MySQL**
4. Убедитесь, что оба модуля горят зеленым

### Шаг 5. Создание базы данных

**Способ A: через phpMyAdmin (рекомендуемый)**
1. Перейдите по адресу: `http://localhost/phpmyadmin`
2. Нажмите **"New"** в левом меню
3. Введите имя базы данных: `chatbot_recruitment`
4. Выберите кодировку: `utf8_general_ci`
5. Нажмите **"Create"**
6. Перейдите во вкладку **"Import"**
7. Выберите файл `chatbot_recruitment.sql` из папки проекта
8. Нажмите **"Import"**

**Способ B: через командную строку MySQL**

```sql
CREATE DATABASE chatbot_recruitment CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
USE chatbot_recruitment;
SOURCE C:/xampp/htdocs/ChatBot-RVK-belozerov/chatbot_recruitment.sql;
```

### Шаг 6. Проверка работоспособности

Откройте в браузере следующие адреса:

| Страница | URL | Назначение |
|----------|-----|-------------|
| Тест подключения | `http://localhost/ChatBot-RVK-belozerov/test.php` | Проверка связи с БД |
| Чат-бот (виджет) | `http://localhost/ChatBot-RVK-belozerov/site.html` | Демонстрация виджета на странице |
| Чат-бот (самостоятельно) | `http://localhost/ChatBot-RVK-belozerov/index.html` | Полноэкранный режим |
| Админ-панель | `http://localhost/ChatBot-RVK-belozerov/admin.php` | Управление вакансиями и заявками |

### Шаг 7. Доступ в административную панель

- **Логин (пароль):** `admin123`
- После успешного входа открывается панель управления со следующим функционалом:
  - Просмотр и управление вакансиями
  - Просмотр заявок кандидатов
  - Фильтрация заявок по вакансиям
  - Статистика по заявкам

## API Эндпоинты

### Публичные эндпоинты (для чат-бота)

| Метод | Эндпоинт | Описание | Пример ответа |
|-------|----------|----------|---------------|
| **GET** | `api.php?action=getVacancies` | Получить список активных вакансий | `[{id, title, description, salary_min, salary_max, schedule}]` |
| **GET** | `api.php?action=getQuestions&vacancy_id={id}` | Получить вопросы для опроса | `[{step_order, bot_message, type, options, field_name}]` |
| **POST** | `api.php?action=submitApplication` | Отправить заявку кандидата | `{"success": true, "id": 88}` |
| **POST** | `api.php?action=uploadResume` | Загрузить файл резюме | `{"file_path": "uploads/xxxxx.docx"}` |

### Закрытые эндпоинты (для администратора)

| Метод | Эндпоинт | Описание | Требования |
|-------|----------|----------|-------------|
| **GET** | `api.php?action=adminGetVacancies` | Получить все вакансии | Сессия администратора |
| **POST** | `api.php?action=createVacancy` | Создать новую вакансию | Сессия + CSRF |
| **POST** | `api.php?action=updateVacancy` | Обновить существующую | Сессия + CSRF |
| **DELETE** | `api.php?action=deleteVacancy&id={id}` | Удалить вакансию | Сессия администратора |
| **GET** | `api.php?action=getApplications` | Получить все заявки | Сессия администратора |
| **DELETE** | `api.php?action=deleteApplication&id={id}` | Удалить заявку | Сессия администратора |

## Структура базы данных

### Таблица `vacancies` (вакансии)

| Поле | Тип | NULL | Описание |
|------|-----|------|----------|
| `id` | INT(11) | Нет | Первичный ключ, AUTO_INCREMENT |
| `title` | VARCHAR(255) | Нет | Название вакансии |
| `description` | TEXT | Нет | Полное описание обязанностей и условий |
| `salary_min` | INT(11) | Да | Минимальная зарплата (руб.) |
| `salary_max` | INT(11) | Да | Максимальная зарплата (руб.) |
| `schedule` | VARCHAR(100) | Да | График работы (5/2, сменный, гибкий, удалённый) |
| `is_active` | TINYINT(4) | Да | 1 - активна, 0 - скрыта |
| `created_at` | TIMESTAMP | Нет | Дата и время создания |

**Пример данных:**
```sql
INSERT INTO vacancies (title, description, salary_min, salary_max, schedule) VALUES
('Слесарь аварийно-восстановительных работ', 'Устранение аварий на сетях водоснабжения...', 45000, 60000, 'Сменный график');
```

### Таблица `applications` (заявки)

| Поле | Тип | NULL | Описание |
|------|-----|------|----------|
| `id` | INT(11) | Нет | Первичный ключ, AUTO_INCREMENT |
| `vacancy_id` | INT(11) | Нет | Внешний ключ → vacancies.id |
| `answers_json` | TEXT | Нет | JSON-объект с ответами кандидата |
| `resume_path` | VARCHAR(255) | Да | Путь к файлу резюме (если загружено) |
| `created_at` | TIMESTAMP | Нет | Дата и время подачи заявки |

**Внешний ключ:**
```sql
ALTER TABLE applications ADD CONSTRAINT fk_applications 
FOREIGN KEY (vacancy_id) REFERENCES vacancies(id) ON DELETE CASCADE;
```

## Валидация данных

### Клиентская валидация (JavaScript)

| Поле | Правило | Сообщение об ошибке |
|------|---------|---------------------|
| **Имя** | Только буквы, кириллица или латиница, минимум 2 символа, первый символ заглавный | "Введите корректное имя" |
| **Телефон** | Маска +7 (XXX) XXX-XX-XX, 11 цифр, первая цифра 7 или 8 | "Введите номер телефона в формате +7" |
| **Возраст** | Число от 18 до 100 | "Возраст должен быть не менее 18 лет" |
| **Гражданство** | Только буквы, автоматическая капитализация | "Используйте только буквы" |
| **Опыт работы** | Число, не может превышать (возраст - 14) лет | "Опыт не может превышать N лет" |
| **Зарплата** | Положительное целое число | "Введите корректную сумму" |

### Серверная валидация (PHP)

- Проверка на пустые значения обязательных полей
- Санитизация имен (удаление спецсимволов)
- Очистка номера телефона от нецифровых символов
- Проверка формата загружаемых файлов (MIME-типы)
- Экранирование HTML-сущностей перед сохранением

### Визуальная обратная связь

При ошибке валидации:
1. Поле ввода подсвечивается красной рамкой
2. Под полем появляется сообщение с описанием ошибки
3. Сообщение автоматически исчезает через 3 секунды
4. Поле сохраняет фокус для исправления

## Скриншоты и описание интерфейса

### Интерфейс чат-бота (виджет)

Чат-бот интегрируется в корпоративный сайт в виде плавающего виджета в правом нижнем углу экрана.

**Структура виджета:**

```

│         РВК-Воронеж                                    [❯]  │  ← Шапка (кликабельная)
│         Чат-бот помощник                                    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Добро пожаловать! Выберите вакансию:                       │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Слесарь аварийно-восстановительных работ            │    │
│  └─────────────────────────────────────────────────────┘    │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Мастер участка водоснабжения                        │    │
│  └─────────────────────────────────────────────────────┘    │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Оператор очистных сооружений                        │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐    │
│  │  Введите сообщение...                               │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**Режимы работы:**
- **Свернутый режим** — виджет отображается только шапкой с логотипом
- **Развернутый режим** — полноценный диалог с ботом
- **Адаптивный** — на мобильных устройствах занимает весь экран

**Процесс диалога:**
1. Бот приветствует пользователя и выводит список доступных вакансий
2. После выбора вакансии показывается её описание и кнопка "Хочу откликнуться"
3. Бот последовательно задает вопросы с валидацией ответов
4. При загрузке резюме отображается имя файла
5. После успешной отправки выводится сообщение и кнопка "Выбрать другую вакансию"

## Возможные проблемы и их решение

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Ошибка подключения к БД** | Неверный порт MySQL | Проверьте порт в `db.php` (по умолчанию 3307) |
| **Пустая страница при загрузке** | Ошибка PHP | Включите `display_errors` в `api.php` |
| **Не загружаются резюме** | Нет папки `uploads/` | Создайте папку вручную и дайте права на запись |
| **Не работает админ-панель** | Сессия не создана | Проверьте настройки session в `php.ini` |
| **Некорректная маска телефона** | Конфликт JavaScript | Обновите страницу и очистите кэш браузера |
| **Вакансии не отображаются** | Таблица пуста | Добавьте вакансии через админ-панель |
