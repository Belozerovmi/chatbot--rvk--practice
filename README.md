## О проекте

Интерактивный чат-бот, разработанный для компании **РВК-Воронеж** в рамках производственной практики. Бот автоматизирует процесс первичного отбора кандидатов на вакансии, собирает необходимую информацию (ФИО, телефон, возраст, гражданство, опыт работы, местоположение, желаемый график, образование, ожидаемую зарплату, а также предоставляет возможность загрузить резюме) и позволяет HR-специалистам управлять вакансиями и просматривать заявки через удобную административную панель.

### Ключевые особенности

- Интеллектуальный опрос — последовательный сбор данных о кандидате с валидацией на стороне клиента и сервера
- Загрузка резюме — поддержка PDF, DOC, DOCX файлов с проверкой MIME-типов
- Административная панель — полное управление вакансиями (CRUD) и заявками с фильтрацией по вакансиям
- Статистика — визуализация общего количества заявок, отфильтрованных заявок и количества уникальных вакансий
- Адаптивный дизайн — корректное отображение на всех устройствах (десктоп, планшет, мобильные телефоны)

## Установка и запуск

### Требования

- XAMPP с PHP 8.0+ и MySQL/MariaDB
- Веб-браузер с поддержкой JavaScript (ES6+)

### Инструкция по установке

1. Клонировать репозиторий в директорию `htdocs` XAMPP:
```bash
git clone https://github.com/your-username/ChatBot-RVK-belozerov.git
cd ChatBot-RVK-belozerov
```

2. Запустить XAMPP Control Panel и включить модули:
   - Apache
   - MySQL

3. Импортировать базу данных:
   - Открыть phpMyAdmin (http://localhost/phpmyadmin)
   - Создать новую базу данных `chatbot_recruitment`
   - Импортировать файл `chatbot_recruitment.sql`

4. Настроить подключение к базе данных в файле `db.php`:
   - Проверить параметры подключения (хост, порт, пользователь, пароль)

5. Открыть проект в браузере:
   - Чат-бот: http://localhost/ChatBot-RVK-belozerov/site.html
   - Админ-панель: http://localhost/ChatBot-RVK-belozerov/admin.php (пароль: admin123)

## API Эндпоинты

| Метод | Эндпоинт | Описание |

| GET | `api.php?action=getVacancies` | Получить список активных вакансий |

| GET | `api.php?action=getQuestions&vacancy_id={id}` | Получить вопросы для вакансии |

| POST | `api.php?action=createVacancy` | Создать новую вакансию |

| DELETE | `api.php?action=deleteVacancy&id={id}` | Удалить вакансию |
| GET | `api.php?action=getApplications` | Получить все заявки |
| POST | `api.php?action=uploadResume` | Загрузить файл резюме |
| DELETE | `api.php?action=deleteApplication&id={id}` | Удалить заявку |

## Валидация данных

Чат-бот выполняет валидацию следующих полей:

| Поле | Правила валидации |
|------|-------------------|
| Имя | Только буквы, минимум 2 символа, первый символ заглавный |
| Телефон | Формат +7 (XXX) XXX-XX-XX, 11 цифр |
| Возраст | От 18 до 100 лет, только цифры |
| Гражданство | Только буквы, автоматическая капитализация |
| Опыт работы | Числовое значение, не превышает возраст - 14 лет |
| Ожидаемая зарплата | Положительное число, только цифры |

## Скриншоты

### Чат-бот (интерфейс пользователя)

Чат-бот представлен в виде виджета, который отображается поверх корпоративного сайта. Виджет может быть свернут и развернут по клику на заголовок.

**Основные элементы интерфейса:**
- Заголовок с логотипом компании и названием
- Область сообщений с прокруткой
- Поле ввода текста с отправкой по Enter или кнопке
- Кнопки быстрого выбора для вариантов ответов
- Индикатор печати при загрузке ответа бота
- Загрузка файлов через стилизованную кнопку

**Процесс взаимодействия:**
1. Бот приветствует пользователя и предлагает выбрать вакансию из списка
2. После выбора вакансии бот показывает её описание и спрашивает о намерении подать заявку
3. При согласии запускается опросник, который последовательно задает вопросы
4. Каждый ответ валидируется, при ошибке показывается сообщение с подсветкой поля
5. После заполнения всех полей заявка отправляется на сервер
6. Бот подтверждает успешную отправку и предлагает выбрать другую вакансию

### Административная панель

Доступна по адресу `/admin.php` с паролем `admin123`.

**Функциональность:**
- Управление вакансиями (создание, редактирование, удаление)
- Просмотр всех заявок с фильтрацией по вакансиям
- Статистика в виде карточек с количеством заявок
- Просмотр загруженных резюме со ссылками на скачивание
- Адаптивный интерфейс с боковым меню

**Особенности админ-панели:**
- Модальные окна для создания/редактирования вакансий
- Подтверждение удаления с предупреждением
- Уведомления о успешных/неудачных операциях
- Выбор графика работы через кнопки-переключатели
